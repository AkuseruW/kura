# Kura

Kura is a Bun-native TypeScript web framework.

## Install

```sh
bun add kura@npm:kurajs
```

Kura applications install the runtime under the local dependency alias `kura`,
so application code can import from `"kura"`.

## Create An App

```sh
bun create kurajs my-app
cd my-app
bun install
bun kura
bun run dev
```
